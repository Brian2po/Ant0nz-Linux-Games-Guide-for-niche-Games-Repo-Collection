# 7 Deadly Sins: Grand Cross (PC Client)

**Status:** 🟢 Playable 

### 📋 Overview
* **Developer/Publisher:** Netmarble
* **Store Link:** [Official PC Client](https://7dsgc.netmarble.com/en/pcclient)
* **Anti-Cheat:** XIGNCODE3 (Confirmed Running)

---

### 🛠️ Installation Steps
This guide covers installation using **Heroic Game Launcher** (recommended).

1. **Initial Setup:** * Open Heroic Game Launcher and click "Run Installer".
   * Select the official `.exe` setup file downloaded from Netmarble.
2. **Installation:** * Follow the Netmarble Launcher installation prompts. 
3. **Configure Path:** * Once installed, you must point Heroic to the actual Launcher executable.
   * **Target Path:** `~/Games/Heroic/Prefixes/default/Netmarble - 7DSGC/drive_c/Program Files/Netmarble/Netmarble Launcher/Netmarble Launcher.exe`
4. **Proton Version:** * Use **Proton-CachyOS** (Latest). 
   * *Tip: You can easily install this using the **ProtonPlus** app.*
5. **Run:** Launch the app, then use the Netmarble Launcher to download the full game data.

### ⚙️ Required Tweaks / Launch Options
* **OS Tested:** Nobara Linux 43 (KDE Plasma)
* **Kernel:** 6.19.5+
* **GPU:** AMD Radeon Vega 8 (Mesa 26.0.1)

### ⚠️ Known Issues & Fixes
* **Anti-Cheat:** XIGNCODE3 is active but currently functions under the specified Proton-CachyOS environment.
* **Paths:** If you use a custom wine prefix, ensure your path in Step 3 matches your specific Heroic directory.
